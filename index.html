SassyWebTemplate

Relies on Bootstrap UI & AngularJS.

Full web template, setup using AngularJS, Bootstrap UI and GruntJS.

Installation

 cd /project/path
 npm install
 grunt

Dependencies

AngularJS

The core of this template is wrapped in an AngularJS App. With a few simple directives included in the lib folder.

Bootstrap UI

This template is built on Bootstrap/AngularJS UI.

GruntJS & NodeJS

While grunt isn't necessarily a dependancy, it's definitely something you should learn. The minified JS and CSS is all generated through Grunt, this allows you to edit your project in a development environment, but only push the live minified code.
