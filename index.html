SassyWebTemplate

Relies on Bootstrap UI & AngularJS.

Full web template, setup using AngularJS, Bootstrap UI and GruntJS.

Installation

 cd /project/path
 npm install
 grunt

Usage

SASS/SCSS/CSS

All files that are css related, are under the css folder. Which has 4 subfolders.

  1. css/scss - If you're capable of editing these files, go for it! It's what you should be editing! Once you've started up grunt, once you make changes to any file in the scss or shared folder, it will update the /main folders files with your output.
  2. css/main - Contains all plugins/addons for this template. You shouldn't really have to touch this, more advanced level user edits.
  3. css/min - The minified file is auto-generated from grunt, whenever a file in the scss/css folder is updated, so is this minified file (if grunt is running).
  4. css/shared - Contains all plugins/addons for this template. You shouldn't really have to touch this, more advanced level user edits.

Angular/JS/Plugins

Files in the js/dest folder are completely automated from grunt, you shouldn't really edit these files unless you never intend to use grunt.

Any files updated in the js/lib folder are automatically merged into the main.min.js file in /js/dest

Only the angular app is compiled into one singular minified file, all angular folders are also merged, but if something isn't merging you'll need to append it to the uglify array in the gruntfile.js file.

Dependencies

AngularJS

The core of this template is wrapped in an AngularJS App. With a few simple directives included in the lib folder.

Bootstrap UI

This template is built on Bootstrap/AngularJS UI.

GruntJS & NodeJS

While grunt isn't necessarily a dependancy, it's definitely something you should learn. The minified JS and CSS is all generated through Grunt, this allows you to edit your project in a development environment, but only push the live minified code.
